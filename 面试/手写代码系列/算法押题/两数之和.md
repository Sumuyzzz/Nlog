描述：给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值** _`target`_ 的那 **两个** 整数，并返回它们的数组下标。


实现思路：hash反向存储
1. 先生成一个哈希列表
2. for循环分别给两个输出的x,y赋值
3. 判断
```js
var twoSum = function(nums, target) {
  let hash = new Map();
  for (let i = 0; i < nums.length; i++) {
    let x = nums[i], y = target - x;
    if (hash.has(y))
      return [hash.get(y), i];
    hash.set(x, i);
  }
};
```
