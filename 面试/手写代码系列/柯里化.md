原理：柯里化实际就是函数转换，将func(a,b,c)改写生func(a)(b)(c),属于一种[[偏函数]]

##### 为什么函数要柯里化（ curring）
实际就是绑定固定参数得到一个偏函数，从而减少参数的复用，本质上是降低通用性，提高适用性。



思路：
1. 根据传入的实际参数，判断是否

##### 简单版
```js
function currying(fn, ...args) {
   return args.length >= fn.length ? fn(...args) : (...args2)=>currying(fn, ...args, ...args2);
}
```
