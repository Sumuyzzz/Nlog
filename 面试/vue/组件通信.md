## 一、props和emit

父组件
```vue
<template>
  <div>
     <h2>父组件区域</h2>
    <hr />
    <Children :num="num"></Children>
  </div>
</template>

<script>
import Children from "./Children.vue";
export default {
  data() {
    return {
      num: 666,
    };
  },
  components: {
    Children,
  },
};
</script>
```

子组件
```vue
<template>
  <div>
    <h2>子组件区域</h2>
   <i>父组件传递过了的值:{{ num }}</i>
  </div>
</template>
<script>
export default {
  props: ["num"],
  data() {
    return {};
  },
};
</script>
```
