### 什么是BFC

翻译过来就是，块级元素上下文；

触发条件：

* 浮动元素
* 绝对定位(position:fixed,absolute;)
* inline block
* overflow
* flex

### css选择器如何确定

1. 选择器越具体，其优先级越高

2. 相同优先级，后面会覆盖前面

3. 属性后加`！important`优先级最高





### 如何清除浮动

1. 给父元素加上`.clearfix`

   ```css
   .clearfix::after{
       content:'';
       display:block;   
       clerr:both;
   }
   ```

   

2. 使用overflow
