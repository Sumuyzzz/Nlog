# 默认不定宽高

### 通用模板

```html
<div class="parent">
  <div class="child">测试文字</div>
</div>
```


### 一、 用`absolute`实现

绝对定位实现垂直居中
1.用translate
```css

.parent{
  position:relative;
  border:1px solid;
}
.child{
  position: absolute;
  top: 50%;
  transform: translateY(-50%);
}

```



### 二、 `flex`

```css

.parent {
  display: flex;
  align-items: center;
  height: 100vh;
}


```


### 三、grid
```css
.parent {
    display: grid;
}
.child {
    justify-self: center;
    align-self: center;
}


```
