知识点：Set,Array.from

Set：会创建一个值不重复的**类数组**

Array.from：将类数组转化为数组

```js
function dedupe(array) {
  return Array.from(new Set(array));
}

dedupe([1, 1, 2, 3]) // [1, 2, 3]

 //简化

const unique = (array)=>[...new Set(array)]


```

Map

```js
function unique (arr) {
    const seen = new Map()
    return arr.filter((a) => !seen.has(a) && seen.set(a, 1))
}
```
