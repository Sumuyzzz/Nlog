# translate

在本来的基础上，自身重新定位也叫**平移**

参数：单个默认x平移，两个参数对应x和y轴

例如：
```css
transform: translate(200px);
```
表示向右平移200px


```css

transform: translateX(200px);
```
以上是`transform: translate(200px);`另一种写法


```css
transform: translate(50%);
```
表示向右平移自身50%的距离

```css
transform: translate(200px,200px);
```
表示向右平移200px,向下平移200px;
